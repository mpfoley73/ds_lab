Build Up a Table

1. The Basic Table

gt() converts a data frame into a gt object.

Show the code 
tbl_1 <- gt::gtcars |>
    head() |>
    select(mfr, model, year, hp, mpg_c, mpg_h) |>
    # STUB
    gt::gt(rowname_col = "mfr") |>
    # TABLE HEADER
    gt::tab_header(
        title = "gt Cars",
        subtitle = "Like mtcars, but in a different package."
    ) |>
    # COLUMN LABELS
    gt::cols_label(
        # mfr = "Manufacturer",
        model = "Model",
        year = "Year",
        hp = "Horsepower",
        mpg_c = "MPG - City",
        mpg_h = "MPG - Hwy"
    ) |>
    # STUB HEAD
    gt::tab_stubhead("Manufacturer") |>
    # TABLE FOOTER
    gt::tab_source_note("gtcars is from the gt package.")

The first thing to notice is that the object has a table body and column labels. It can potentially have a stub (row groups or labels), header, and footer.

This is analagous to the ggplot object. There is a base plot with x and y aesthetic definitions and labs(). The analogy is useful because “gt” stands for “grammar of tables” - a loose parallel to “grammar of graphics”.1

Grammar of Tables

The Grammar of Graphics is a framework of building up a plot with layers. The plot aesthetics map the table attributes to plot attributes. The code does not specify ‘draw this here’, but rather ‘apply these rules for this geom.’

The Grammar of Tables is similar in that the pipeline of gt functions applies rules for how to treat the data frame.

2. Footnotes

You can add footnotes almost anywhere on the table. Base command is tab_footnote(footnote, locations) where locations is a function related to a part of the table:

  • cells_body()
  • cells_column_labels()
  • cells_column_spanners()
  • cells_grand_summary()
  • cells_group()
  • cells_row_groups()
  • cells_source_notes()
  • cells_stub()
  • cells_stub_grand_summary()
  • cells_stub_summary()
  • cells_stubhead()
Show the code 
tbl_2 <- tbl_1 |>
    gt::tab_footnote(
        "Miles per gallon",
        locations = gt::cells_column_labels(columns = mpg_c)
    ) |>
    # Footnote in table body. Uses formula.
    gt::tab_footnote(
        "Fuel efficient!",
        locations = gt::cells_body(
            columns = mpg_h,
            rows = mpg_h > 20
        )
    )
